# Ngoato Mogoshadi Lodge Website

A modern, responsive website for Ngoato Mogoshadi Lodge in Lebowakgomo.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Gallery**: Click on images to view them in full-screen lightbox mode
- **Integrated Google Maps**: Shows the exact location with directions
- **Custom Logo**: Creative SVG logo representing the lodge
- **Smooth Animations**: Elegant scroll animations and transitions
- **Contact Form**: Easy booking inquiry form
- **Modern UI**: Clean, professional design with gold and earth-tone color scheme

## Files

- `index.html` - Main website structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and functionality

## How to Use

1. Open `index.html` in any modern web browser
2. The website is fully functional and ready to deploy
3. Gallery images are sourced from Unsplash (can be replaced with actual lodge photos)

## Sections

1. **Hero Section**: Eye-catching landing area with call-to-action
2. **About**: Information about the lodge
3. **Amenities**: Key features and services
4. **Gallery**: Photo gallery with lightbox viewer
5. **Location**: Interactive Google Maps integration
6. **Contact**: Booking inquiry form

## Deployment

To deploy this website:
1. Upload all files to your web hosting service
2. Ensure all files are in the same directory
3. Point your domain to the hosting location
4. The website will be live!

## Customization

- Replace gallery images with actual photos of the lodge
- Update the color scheme in `styles.css` (CSS variables in `:root`)
- Add more amenities or information as needed
- Connect the form to a backend service for email notifications
